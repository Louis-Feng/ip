When you run the program(Level 9), you should see
____________________________________________________________

 Hello! I'm Duke

 What can I do for you?

____________________________________________________________
you can type your command and Duke will help you
1. list to show all your tasks to do
2. done (number) you have done this task and mark this as done
3. todo (things to do) create a todo task
4. delete (number) delete the corresponding task
5. find (keyword) find all the tasks containing this keyword
6. deadline (task / by time) set a deadline task which needs to be done by a certain time
7. event (task / at time) set an event task which will happen at a certain time
8. bye  quit the program 
What is inside brackets should be the input format!
